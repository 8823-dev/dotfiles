## Settings

*mi > アプリケーション設定*

- 一般
  - フォント
    - 源ノ角ゴシック Code JP R, 36.0 pt
- 表示
  - ウインドウのレイアウト
    - 右サイドバーを表示しない
- 色
  - 常にダークモード（mi-dark）
- 文字挿入位置と現在行
  - キャレット色を指定
    - 適当に見やすい色。これは未設定でもいいかも
